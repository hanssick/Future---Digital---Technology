# 🌐 Digital Future Technology - Hub de Infrastructură Digitală

Proiect web modern, complet optimizat (Responsive), dezvoltat pentru prezentarea unei firme de servicii IT, rețelistică și soluții de televiziune prin satelit. Site-ul integrează cerințe tehnice avansate de layout, accesibilitate și design.

## 🚀 Tehnologii Utilizate

* **HTML5 Semantic:** Structură clară folosind tag-uri moderne (`header`, `main`, `section`, `footer`, `hgroup`).
* **CSS3 Advanced:** * **Layout:** CSS Grid (`grid-template-areas`) și Flexbox.
    * **Design:** Variabile CSS (`:root`), animații, tranziții și efecte de profunzime (`box-shadow`).
    * **Responsiveness:** Mobile-first approach cu Media Queries pentru desktop, tabletă și mobil.
* **JavaScript:** Logică pentru interactivitate (coș de cumpărături, schimbare dinamică video).
* **MathML:** Integrarea formulelor matematice complexe pentru calculul câștigului antenelor.
* **Font Awesome & Google Fonts:** Iconografie profesională și tipografie externă via API.

## 🛠️ Caracteristici Principale

### 1. Layout Inteligent (Grid & Flex)
Site-ul folosește un sistem de grid asimetric pe desktop, care se reorganizează fluid pe dispozitive mobile. Zonele sunt izolate vizual prin umbre și margini rotunjite, respectând principiile de ierarhie vizuală.

### 2. Design "Dark Mode"
O schemă cromatică din 5 culori bazată pe nuanțe de gri antracit și albastru electric, optimizată pentru a reduce oboseala oculară și a inspira profesionalism tehnologic.

### 3. Componente Interactive
* **Sistem de Taburi Iframe:** Navigare fluidă între tutoriale video fără reîncărcarea paginii.
* **Coș de Cumpărături:** Funcționalitate JS pentru adăugarea abonamentelor și calculul totalului.
* **Tabel Avansat:** Randuri cu stilizare alternantă, efecte de hover (zoom 20% pe text) și scroll orizontal pe ecrane mici.
* **Buton "Back to Top":** Creat 100% în CSS (fără imagini), cu formă de romb și con, dotat cu tooltip informativ.

### 4. Optimizare Media
Utilizarea tag-ului `<picture>` pentru servirea imaginilor în funcție de rezoluție și integrarea documentației tehnice direct în pagină via `<embed>` (PDF).

## 📂 Structura Fișierelor

* `index.html` - Conține structura semantică și logica proiectului.
* *Notă: Stilul și scripturile sunt integrate în fișierul principal pentru portabilitate maximă, dar respectă separarea logică prin comentarii și variabile.*

## 📈 Principiile de Design Aplicate

* **F-Pattern:** Optimizarea citirii în zona de prezentare prin ancore vizuale stângi.
* **Legea Proximității:** Gruparea logică a elementelor de acțiune (butoane lângă prețuri).
* **Scalabilitate Tipografică:** Utilizarea unităților relative (`rem`, `em`) corelate cu un font-size dinamic pe `html`.
